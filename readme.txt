# TestsCreation
Класс с тестами на класс `Rational`, которые проверяют создание экземпляра класса или вызов сеттеров. 

Тест void testStandardConstructor() проверяет значения числителя и знаменателя стандартного экземпляра класса.
Тест void testConstructorWithParameters() проверяет значения числителя и знаменателя произвольного экземпляра класса.
Тест void ConstructorWithParametersNegative() проверяет вызов исключения при создании дроби с нулевым знаменателем.
Тест void testToString() проверяет правильность перевода дроби в строку.
Тест void testReduce() проверяет правильность сокращения дроби (правильность нахождения НОД, работы со знаком "-").
Тест void SetDenominatorNegative() проверяет вызов исключения при изменении знаменателя дроби на нуль.

# TestsOperation
Класс с тестами на класс `Rational`, которые проверяют арифметические операции с экземплярами класса. 

Тест void testPlus() проверяет правильность выполнения операции сложения рациональных чисел.
Тест void testMinus() проверяет правильность выполнения операции вычитания рациональных чисел. 
Тест void testMultiply() проверяет правильность выполнения операции умножения рациональных чисел. 
Тест void testDivide() проверяет правильность выполнения операции деления рациональных чисел. 
Тест void DivideNegative() проверяет вызов исключения при делении на дробь с нулевым числителем.

# TestsComparison
Класс с тестами на класс `Rational`, которые проверяют операции сравнения экземпляров класса. 

Тест void testEquals() проверяет правильность выполнения операции сравнения "равно".
Тест void testLess() проверяет правильность выполнения операции сравнения "меньше".
Тест void testLessOrEqual() проверяет правильность выполнения операции сравнения "меньше или равно".
